# 放課後の情報室からの脱出

PythonとFletで作成した、シンプルな謎解き脱出ゲームです。

## 概要

プレイヤーは放課後の情報室に閉じ込められています。机、黒板、パソコン、本棚、ロッカー、プリンタ、ドアを調べ、複数の手がかりを組み合わせて脱出を目指します。

## 実行方法

```bash
pip install -r requirements.txt
python main.py
```

この環境では仮想環境 `.venv` を作成済みです。すぐ起動する場合は以下を使用できます。

```bash
source .venv/bin/activate
python main.py
```

`pip` がない環境では、`uv` を使ってセットアップできます。

```bash
uv venv .venv
uv pip install -r requirements.txt
source .venv/bin/activate
python main.py
```

## 操作方法

- 「ゲーム開始」で情報室に入ります。
- 見取り図の机、黒板、パソコン、本棚、ロッカー、プリンタ、ドアを押して調べます。
- パスワードやキーワードが必要な場面では、表示された入力欄に答えを入力します。
- 係メモ、座席表、色ログなどを組み合わせて謎を解きます。
- 小さな鍵、ドアカード、最終メモを手に入れ、最後の暗証番号を入力すると脱出成功です。
- 行き詰まった場合は、所持品欄の電球アイコンでヒントを確認できます。
- 「もう一度遊ぶ」または「最初から」で最初からやり直せます。
