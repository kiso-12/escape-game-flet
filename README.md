# 放課後の校舎からの脱出

PythonとFletで作成した、シンプルな謎解き脱出ゲームです。

## 概要

プレイヤーは放課後の情報室から始まり、廊下、準備室、昇降口へ移動しながら校舎からの脱出を目指します。机、黒板、掲示板、パソコン、本棚、ロッカー、プリンタ、分電盤、非常口端末などを調べ、複数の手がかりを組み合わせます。

行動、ヒント、不正解で集中力が下がります。集中力が0になると先生が戻ってきてゲームオーバーです。クリア時には、行動回数、残り集中力、ヒント使用回数、不正解回数から評価ランクが表示されます。

## 実行方法

```bash
pip install -r requirements.txt
python main.py
```

この環境では仮想環境 `.venv` を作成済みです。すぐ起動する場合は以下を使用できます。

```bash
source .venv/bin/activate
python main.py
```

`pip` がない環境では、`uv` を使ってセットアップできます。

```bash
uv venv .venv
uv pip install -r requirements.txt
source .venv/bin/activate
python main.py
```

## 操作方法

- 「ゲーム開始」で情報室に入ります。
- 情報室、廊下、準備室、昇降口を移動しながら調べます。
- 見取り図の机、黒板、掲示板、パソコン、本棚、ロッカー、プリンタ、ドア、階段、分電盤、非常口端末などを押して調べます。
- パスワードやキーワードが必要な場面では、表示された入力欄に答えを入力します。
- 係メモ、座席表、巡回メモ、色ログ、避難経路図などを組み合わせて謎を解きます。
- 情報室のドアを開けた後も、廊下と準備室で手がかりを集め、最後に昇降口から建物の外へ出ると脱出成功です。
- 行き詰まった場合は、所持品欄の電球アイコンでヒントを確認できます。
- ヒントや不正解は集中力を消費するため、使いすぎには注意が必要です。
- 「もう一度遊ぶ」または「最初から」で最初からやり直せます。
